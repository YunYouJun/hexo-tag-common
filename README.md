# hexo-tag-common

Common tags for hexo.

> Demo: [tag-common | yun](https://www.yunyoujun.cn/yun/tag-common.html)

## Config

Set it in hexo `_config.yml`.

```yaml
tag_common:
  cdn: true # enable jsdelivr CDN
```
